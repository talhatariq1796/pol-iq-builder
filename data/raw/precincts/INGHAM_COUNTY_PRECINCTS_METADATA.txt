FIELD STRUCTURE
---------------
- NAME: Unique precinct identifier (e.g., "0650080000001")
- COUNTYFIPS: County FIPS code ("065")
- MCDFIPS: Minor Civil Division FIPS code
- Precinct_Long_Name: Full precinct name (e.g., "Alaiedon Township, Precinct 1")
- Precinct_Short_Name: Abbreviated name (e.g., "Alaiedon Twp 1")
- Jurisdiction_Name: Municipality name

DATA QUALITY NOTES
------------------
The 2024 Precinct data set represents the geography used for the 2024 election cycle.
Information was collected from local election officials along with county/local GIS 
authorities as well as a validation done by most, but not all jurisdictions.
